## Build Setup

``` bash
# install dependencies
$ pipenv install -d

# serve with hot reload at http://localhost:8000
$ mkdocs serve

# generate static project
$ mkdocs build
```

For detailed explanation on how things work, checkout [MkDocs docs](https://mkdocs.org).
